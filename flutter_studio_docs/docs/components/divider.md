# Divider

Visual content separator.

## Installation

::: code-group

```bash [CLI]
flutter_studio add divider
```

```bash [Package]
flutter pub add flutter_studio
```

:::

## Usage

```dart
CustomDivider()

// With label
CustomDivider(
  label: 'OR',
)
```
